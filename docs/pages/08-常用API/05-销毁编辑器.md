# 销毁编辑器

执行 `editor.destroy()` 可以销毁编辑器实例。在 Vue2.x 中使用销毁 API ，代码如下。

```js
// 用于 Vue2.x
import E from '@wangeditor-team/we-next'

export default {
  data() {
    return {
      editor: null
    }
  },
  mounted() {
    // 创建编辑器
    this.editor = new E(`#demo`)
    this.editor.create()
  },
  beforeDestroy() {
    // 销毁编辑器
    this.editor.destroy()
    this.editor = null
  }
}
```
